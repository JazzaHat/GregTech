Crafting Recipes

  1. Create the base recipe image file first. Photoshop (or use any other free image editor, such as GIMP) the item images onto a crafting grid template.
    1. Templates
      1. Shaped Crafting
      2. Shapeless Crafting
    2. You can obtain images of GregTech items from gregtechmod.zip/assets/gregtechmod. However, they are 16x16, so you should enlarge them to 32x32 first with no interpolation.
      1. Refer to config/GregTech.lang for the items labeled with numerals
    3. You can obtain items of other things by hitting F2 ingame. However, they are 48x48, so you should resize them to 32x32 with no interpolation.
    4. The convention is to name it item_name_recipe.png
      Example:
      digital_chest_recipe1.png
    5. If the recipe yields multiple items (e.g. Data Orbs), please use "Minecraftia" font and it should be in the bottom-right corner, moved to the centre by 2x2 pixels.
  2. Upload your base recipe image file to the Wikispace
  3. Afterwards, create a notepad file (you don't need to save it yet) with the following HTML code
    <img src=[1] usemap=[2]>
    1. [1] is the link to the recipe image file. Find it using the Pages and File search function, click on the link beside the filename. Afterwards, copy the link address, but omit the "gregtechcommunitywiki.wikispaces.com" portion. Enclose it in quotation marks.
      Example:
      "file/view/new-AESU-crafting.png/466544084/new-AESU-crafting.png"
    2. [2] is a string of words you create. The wiki convention is to name it "item_name_recipe".
      Example:
      "digital_chest_recipe1"
    3. "aesu_recipe_new"
  4. Add the following code:
    <map name=[2]>
[2] should be the name you chose previously.

  1. Add the following line of code for every input and output slot used
    <area shape="rect" coords=[3] href=[4] title=[5]>
    1. [3] depends on the location of the item. See the appendix.
    2. [4] is the URL you want to connect to, enclosed in quotation marks. Omit this portion if there is no wiki page for the item yet.
    3. [5] is the name of the item. Enclose it in quotation marks.
    4. Example:
      <area shape="rect" coords="34,106,66,138" title="Energy Flow Circuit">
<area shape="rect" coords="70,34,102,66" href="GregTech+ComputerCube" title="GregTech ComputerCube">
  2. Finally, end the code with
    </map>
  3. Create a widget in the space you want to add the recipe. Select the type "Other" or "Other HTML".
  4. Copy and paste your code into the space provided. Hit preview to check that it is what you want.
    Example:
    <img src="file/view/new-AESU-crafting.png/466544084/new-AESU-crafting.png" usemap="aesu_recipe_new">
<map name="aesu_recipe_new">
<area shape="rect" coords="34,34,66,66" title="Energy Flow Circuit">
<area shape="rect" coords="34,70,66,102" title="Energy Flow Circuit">
<area shape="rect" coords="34,106,66,138" title="Energy Flow Circuit">
<area shape="rect" coords="70,34,102,66" href="GregTech+ComputerCube" title="GregTech ComputerCube">
<area shape="rect" coords="70,70,102,102" href="Lapotronic+Energy+Orb" title="Lapotronic Energy Orb">
<area shape="rect" coords="70,106,102,138" title="Highly Advanced Machine Block">
<area shape="rect" coords="106,34,138,70" title="Energy Flow Circuit">
<area shape="rect" coords="106,70,138,102" title="Energy Flow Circuit">
<area shape="rect" coords="106,106,138,138" title="Energy Flow Circuit">
<area shape="rect" coords="222,70,254,102" href="AESU" title="AESU">
</map>
  5. When you have corrected all errors, click "Submit".
  6. Hit "Save" in the editor.

Appendix

URL Tips

  • If you are trying to link to a page in GTCW, you can omit
    gregtechcommunitywiki.wikispaces.com/

Value of Coords

Crafting Recipe

  • Vertical
    • Top row: XX, 34, XX, 66
    • Middle row: XX, 70, XX, 102
    • Bottom row: XX, 106, XX, 138
  • Horizontal
    • Left column: 34, XX, 66, XX
    • Middle column: 70, XX, 102, XX
    • Right column: 106, XX, 138, XX
  • Output
    • 222,70,254,102
  • Remember to enclose your value in quotation marks.
