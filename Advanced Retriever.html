The Advanced Retriever is able to pull items from an inventory using GregTech Pipes, and place them into another inventory based on the slot and stack size. This was introduced in 4.07w.


Tool Tip:

  • Retrieves and Regulates Items through a GT Pipe System!
  • Default Max: EU/t 128
  • Possible Upgrades: T B M S


GUI:


  • Item Filter/Stack Size:
    • Left/Right click: Decrease/Increase stack size by 1.
    • Shift + Left/Right: Decrease/Increase stack size by 8.
  • Ouput Inventory Slot Number:
    • Left/Right click: Decrease/Increase slot # by 1.
    • Shift + Left/Right: Decrease/Increase slot # by 16.
  • Partial Request Allowed: This means that it will pull matching filtered items until the stack size is reached.
  • Partial Request Forbidden: This means that it will only pull matching filtered items if there is enough to reach the filtered stack size.
