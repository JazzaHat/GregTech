BlockItemIDs.cfg is pretty much exactly what its name says; this file contains all the ID numbers used by all blocks and items added by GregTech. If you have a block or item ID conflict that throws a crash, this is where you could fix it on the GregTech end.
The following is a freshly generated copy+paste of BlockItemIDs.cfg, from a Minecraft 1.5.2 instance running GregTech 3.11.
  • All lines with bullet points, like this line, are comments added to clarify the lines in the configuration file. They are not part of the acutal file.

# block
####################
block {
I:Block=4058
  • Material storage blocks (Lead, Electrum, Zinc, Olivine, Green Sapphire, Platinum, Tungsten, Nickel, Tungstensteel, Invar, Osmium, Iridium), and Iridium-Reinforced Tungstensteel
  • Mobs cannot spawn on these blocks
I:Block2=4057
  • Metal storage blocks (Silver, Ruby, Sapphire, Aluminium, Titanium, Chrome, Steel, Brass), Advanced and Highly Advanced Machine Blocks, LESU-Blocks (depricated), Machine Casings, Iridium Reinforced Stone, and Fusion Coils
  • Mobs cannot spawn on these blocks
I:LightSource=4061
  • This is currently unused, but is generated anyway. GregTech 4.02h actually does add a Redstone Lamp, but that is a 1.6.2 version.
I:Machine=4059
  • ALL GregTech machinery blocks
  • All blocks in this ID with damage values lower than 16 (4059:2 thru 4059:15) are older (GregTech 2.0, MC 1.4.7 and before) machines. At the end of GT 2.0, MC 1.4.7, almost all GregTech machines were converted from TileEntities to MetaTileEntities, and many of their attributes were standardized. This is what allowed for the Upgrade and Cover systems, and for all machines to occupy only 1 block ID.
I:Ore=4060
  • GregTech Ore blocks
}

####################
  • GregTech adds several new Crops to expand the IndustrialCraft2 crop system. Good luck breeding all of them.
  • There is a Book (Crop Dictionary by Mr. Kenny) that can be found in dungeon chests, that explains some of the aspects of each of these plants.
  • This section probably shouldn't need to be messed with, unless you have other mods that add new IC2 Crops, and there is a conflict between them.

# item
####################
  • There are a lot of items added by GregTech. Any items which have multiple variants, including variants added by other mods, have their names in ALL CAPITAL LETTERS (like CELLS full of various liquids), and consume only 1 ID and a large number of damage values to represent a large number of different items.
  • COMPONENTS are Circuits, Superconductors (item), Covers, Duct Tape, Sawblades, Grinders, Heating Coils, Upgrades, Circuit Boards, Machine Hulls, Empty Spray Cans, Turbine Blades, and Gears
  • MATERIALS are Credits, Ingots, Oil Berries, Indigo, Flour, Plates, Gems, and Metal Rods
  • Unlike the other ID sections, all of these IDs are 256 higher in-game.
