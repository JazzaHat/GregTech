Section 1: Installing Minecraft Forge

Minecraft 1.6.x onwards

  1. Download your preferred version of Minecraft Forge from here. Click the "installer" link. Here are some quick links:
    Stable Latest
    1.6.2
    #871
    1.6.4
    #965
  2. Run the installer. On Windows, right-click the .jar file and select "Run with Java(TM) Platform SE binary". Select "Install client" and direct it to where you installed Minecraft.
  3. Wait for the installer to download required libraries and install them.
  4. Run Minecraft and close it.

Minecraft 1.3.2 - 1.5.2

  1. Download your preferred version of Minecraft Forge from here. Click the "universal" link. Here are some quick links.
    1.3.2: #318
    1.4.2: #355
    1.4.5: #448
    1.4.6: #489
    1.4.7: #534
    1.5.2: #737 #738
  2. Find your minecraft.jar file. On the default launcher in Windows, run %AppData%/.minecraft/bin
  3. Open minecraft.jar with WinRAR or 7zip in Windows. On Mac, extract minecraft.jar to a folder.
  4. Drag-and-drop the contents of Minecraft Forge zip file into minecraft.jar. On Mac, afterwards recreate minecraft.jar
  5. Run Minecraft and close it.

Section 2: Installing Other Mods

  1. In your Minecraft installation folder, you should find a folder called mods.
  2. Drag-and-drop the mod file (i.e. gregtechmod.zip) into mods folder.
