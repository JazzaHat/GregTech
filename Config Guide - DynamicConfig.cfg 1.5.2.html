DynamicConfig.cfg is the longest and most complicated GregTech config file. This file contains settings for every single recipe added by the mod, and several other more optional or peripheral functions. It is Dynamically generated based on what other mods you have active.

The following is a freshly generated copy+paste of DynamicConfig.cfg, from a Minecraft 1.5.2 instance running GregTech 3.11.
  • All lines with bullet points, like this line, are comments added to clarify the lines in the configuration file. They are not part of the acutal file.
Many lines in this file are related to one another. Comments will be added after sets of related lines.
