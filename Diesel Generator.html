Usage Tips

Note that while the internal storage is stated (via Scanner) to be 1 million, the generator will not attempt to generate EU if it would push its stored energy above 1000.

Block tip usage : Forestry Ethanol and MFR biofuel are abundant if you make a setup to produce them. MFR biofuel no longer produces any energy in the Diesel generator.
MFR Biofuel can be turned into Forestry biofuel by using MFR unifier. Input MFR biofuel, output Forestry ethanol

MFR = Minefactory Reloaded
