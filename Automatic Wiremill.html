"Produces Wires more efficiently"
The GregTech Intergalactical Automatic Wiremill is a relatively low-tech machine that can be used to increase the material efficiency at which IC2 metal Cables can be produced. It is also used to process the Soldering metals, used by the Soldering Iron.
Since 4.04a Heating Coils are crafted in a wiremill, using alloyed ingots, instead of a rolling machine.
