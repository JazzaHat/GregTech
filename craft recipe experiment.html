I have simply cut and pasted screenshots around to get the GUI templates. They may be slightly off-center, awkward, or otherwise have problems. Feel free to change them; however, understand that you will have to adjust the coordinates appropriately for every instance of that recipe used in the wiki.

All Recipes

1. You need a picture of the recipe. You can obtain item recipe components from the gregtechmod.zip/mods/GregTech_Addon/textures. For blocks, you can find it by copying and pasting parts of GT_Recipes.png. Or, if the block is not found there, take an NEI screenshot with GUI Scale: Normal.
2. See Appendix A for templates to use as the HTML coding below is based off these templates.
3. Use an image editor that supports transparencies (e.g. GIMP, Paint.NET, Photoshop). MS Paint does NOT support transparencies and will not work properly.
4. Move the item images to the correct slots to form the recipe.
5. Upload them to this wiki. My convention is [ITEM_NAME]_[MACHINE_NAME].png. [MACHINE_NAME] is crafting, smelting, maceration, assembly, bending, etc.
6. Copy this code to a text editor. (e.g. Notepad, Wordpad, Notepad++)
<img src="file/view/[NAME_OF_YOUR_RECIPE].png" height="172" width="296" usemap="[DISTINCT_NAMING]"/>
Remember to replace anything in brackets with the correct term. For the [DISTINCT_NAMING], my convention is [PAGE_NAME]_crafting_[NUMBER]. If the crafting recipe is the 1st in a page, put 1 for [NUMBER]. If it is 2nd, put 2. And so on.
7. Continue the code. [DISTINCT_NAMING] must be the same as the one used in step 6.
<map name="[DISTINCT_NAMING]" id="[DISTINCT_NAMING]">
8. Continue the code. Repeat the following for every slot used in the recipe. Use the table in Appendix B to know what to put for [COORDS]. [PAGENAME_OF_MATERIAL] must be the name of a page for the item in that recipe slot. If there is no page for the item, leave it blank between the quotation marks. [NAME_ON_MOUSEOVER] is the name of the item.
<area coords="[COORDS]" href="[PAGENAME_OF_MATERIAL]" alt="[NAME_ON_MOUSEOVER]" title="[NAME_ON_MOUSEOVER]"/>
9. Finally end the code with
</map>
10. Create an "Other HTML" widget in the page you want to edit and copy the code in your text editor into the space in the widget. Click "Preview" to check you've done things right, then click "Submit". "Preview" will not show the widget properly until the page has been saved with at least one recipe widget.
