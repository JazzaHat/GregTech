"Steam Powered Extractor"

*1.6.2 machine*

Sticky Resin can no longer be cooked in a Furnace to produce Rubber. To create the Rubber required for construction of your first Electric Machines, you will need a Squeezing Extractor. The Squeezing Extractor turns certain items into Rubber Pulp, which is then turned into rubber by smelting it in a furnace.

This Extractor functions the same as the Automatic Extractor, but operates on Steam rather than EU.
Steam for this machine is most likely to be supplied by a Small Coal Boiler or a High-Pressure Steam Boiler.

For reference, obtaining Rubber in the Squeezing Extractor can be done by:

1 x Rubber Sapling/Rubber Wood --> 1 x Rubber Pulp
1 x Slime Ball --> 2 x Rubber Pulp
1 x Sticky Resin --> 3 x Rubber Pulp

The Extractor requires to be supplied Steam via piping to any of it's sides. However, be aware that this machine, as with all Steam Machines, has a square Vent which will eject a jet of steam after each work is finished, the adjacent tile to this Vent needs to be unobstructed in order for the machine to keep operating. Also, it'll burn you if you're there. Steam vent can be adjusted using a Wrench.
A common problem with Steam machines is that they will run one process, then inexplicably stop. This problem is caused by the Vent being obstructed.

Recipe:




5 x Bronze Plate
2 x Piston / Sticky Piston
1 x Aluminium/Iron/Bronze/Brass Machine Hulls OR Machine Box OR Machine Block
1 x Wrench
