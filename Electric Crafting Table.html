First of all this is an autocrafting table, with several modes to allow better and easier usage of the block.
It will receive items from the green side (buffers, translocators or conveyor cover) and try to craft them into something, then output to the inventory adjacent to the red side.
This block has an integrated unification operation, making items to automatically output only one kind of item, even if several different mod items are inserted.
For example you can insert any mod tin and copper dust and you will get ICÂ² bronze dust, same for electrum and other items.

Dust and nugget combining modes have reduced operation cost.
It has an internal tank to fill containers for craftings that requires filled containers.
All cells on recipes will get recycled (turned into empty cells for re-usage).
Accepts covers and upgrades.
Accepts liquids from liquiducts or pipes.

Maximum EU input : 32 EU/p , Transformer upgrades quadruples this value.
Maximum EU output : 32 EU/t, Transformer upgrades quadruples this value. First transformer upgrade will make it output 4 packets instead of quadrupling the packet size
