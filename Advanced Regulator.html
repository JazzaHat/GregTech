The Advanced Regulator is an expensive and powerful machine. It is able to control the stack size of 9 items input items into a machine. This is extremely useful for the Industrial Electrolyzer and Industrial Centrifuge, both which have multiple recipes that have varying stack requirements.

Tool Tip:

  • Lets you save up to 8 additional Machines when used correctly!
  • Default Max: EU/t 128
  • Possible Upgrades: T B M S


GUI:


  • Item Buffer: Items will wait here until filter & stack size requirement has been met.
  • Item Filter/Stack Size:
    • Left/Right click: Decrease/Increase stack size by 1.
    • Shift + Left/Right: Decrease/Increase stack size by 8.

  • Ouput Inventory Slot Number:
    • Left/Right click: Decrease/Increase slot # by 1.
    • Shift + Left/Right: Decrease/Increase slot # by 16.
