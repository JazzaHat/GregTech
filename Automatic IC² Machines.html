GregTech overwrites the standard IC² machine recipes with the GregTech Automatic Machines. GregTech Automatic Machines have exactly the same crafting recipes and functions as their non-automatic IC² counterparts, but will automatically output their products (and, if enabled, EU) through the square on their backsides into an adjacent inventory. The output side can be re-positioned using a Wrench. Additionally, GregTech Automatic Machines have two input slots and two output slots, and are compatible with the GregTech Upgrade and Cover systems.

Possible Upgrades:
Overclocker
Transformer
Battery
Minecraft Joules (MJ)
Steam
Maximum EU/p In:
32 (LV)

Maximum EU/t Out (toggleable):
32 (LV)


If you want to use vanilla IC² machines, you must go to the gregtechrecipes section of Dynamic Config.cfg, and disable the option, "B:automaticmachines=" (true by default), which will disable the Automatic Machine recipes and return the vanilla IC² machines, OR enable the option "B:allowVanillaIC2MachinesFromAutomaticOnes=" (false by default), which allows you to shapelessly craft standard IC² machines from their GregTech Automatic counterparts.
