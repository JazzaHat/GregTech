Very complicated machine which allows compact automation setups.
Here is a picture explaining its GUI and how it works :

Remember that if you only want one item to be imported/extracted to a colored face, you must block other item filters of the same face with random stuff.
For example if i want to only import glass fiber cables from the magenta side i would need to place other two items in the two other filters, so the magenta face wont import anything else but glass fiber cables.

Accepts covers and upgrades.
Input up to 128 EU packets.
Output energy on allowed sides at a maximum rate of 128 EU/t, within four 32 EU packets, if two or more sides are allowed the maximum rate will be split.

Output rate/ packets outputted/ maximum packet size input increases with every transformer upgrade in this manner :
128/4x32/128 (0) -> 128/4x32/512 (1) -> 512/4x128/2048 (2) -> 2048/4x512/8192 (3, max)
